Project 01

Student Management System

A web-based application to manage student records efficiently with full CRUD operations.

Web Application HTML · CSS · JavaScript Completed

Overview

The Student Management System is a fully browser-based application that lets administrators manage student data without a backend. All data is stored locally using the browser's LocalStorage API.

Features

  • Add new student records with name, roll number, grade, and contact info
  • View all students in a searchable table
  • Edit existing student details inline
  • Delete individual records with a confirmation step
  • Persistent data via LocalStorage — no refresh loss

Tech Stack

HTML5 CSS3 JavaScript ES6 LocalStorage API

Project 03

To-Do List Application

A lightweight task manager that keeps your daily goals organised with persistent storage.

Web App HTML · CSS · JavaScript Completed

Overview

A minimal, fast task manager built in the browser using vanilla JavaScript. Tasks persist between sessions via LocalStorage — no login or account needed.

Features

  • Add tasks instantly with a click or pressing Enter
  • Mark tasks as complete with a checkbox toggle
  • Delete individual tasks or clear all completed ones
  • Filter by All / Active / Completed
  • Persistent data via LocalStorage

Tech Stack

HTML5 CSS3 JavaScript ES6 LocalStorage API

Project 04

Weather Forecast App

Search any city and get live temperature, humidity, and weather conditions instantly.

Web App JavaScript · REST API Completed

Overview

The Weather Forecast App uses the OpenWeatherMap API to fetch real-time weather for any city in the world. Users see current temperature, humidity, wind speed, and a short-term forecast instantly.

Features

  • City search with live data from OpenWeatherMap
  • Temperature, humidity, and wind speed display
  • Celsius / Fahrenheit toggle
  • 5-day forecast summary
  • Error handling for invalid city names

Tech Stack

HTML5 CSS3 JavaScript ES6 OpenWeatherMap API Fetch API
